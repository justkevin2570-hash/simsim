수학적 귀납법

자연수 명제를 증명하는 강력한 도구 Mathematical Induction

🧠 개념 이해

수학적 귀납법은 모든 자연수 \(n\)에 대해 어떤 명제 \(P(n)\)이 참임을 증명하는 방법입니다. 마치 도미노가 연쇄적으로 넘어지듯, 첫 번째가 참이고, 앞의 것이 참이면 다음 것도 참임을 보여 전체가 참임을 증명합니다.

📌 핵심 아이디어
"첫 번째 도미노가 넘어진다" + "k번째가 넘어지면 k+1번째도 넘어진다"
→ 모든 도미노가 넘어진다! 🎯

🎲 도미노 비유

도미노가 연쇄적으로 넘어지는 모습을 통해 귀납법의 원리를 시각화합니다

준비 — 첫 번째 도미노를 넘겨보세요

📋 증명의 3단계

수학적 귀납법은 항상 아래 세 단계로 구성됩니다.

1️⃣

기초 단계 (Base)

\(n=1\) (또는 시작값)일 때 명제 \(P(1)\)이 참임을 확인한다.

\(P(1)\) is true
2️⃣

귀납 가정 (Hypothesis)

\(n=k\)일 때 명제 \(P(k)\)가 참이라고 가정한다.

\(P(k) \implies\) 가정
3️⃣

귀납 단계 (Inductive Step)

\(P(k)\)가 참이면 \(P(k+1)\)도 참임을 증명한다.
즉, \(P(k) \Rightarrow P(k+1)\)

\(P(k) \Rightarrow P(k+1)\)
✅ 결론: 기초 단계 + 귀납 단계가 증명되면,
모든 자연수 \(n\)에 대해 \(P(n)\)은 참이다!
\(P(1)\) 참 → \(P(2)\) 참 → \(P(3)\) 참 → ... → \(P(n)\) 참

📐 예제: 자연수의 합

다음 명제를 수학적 귀납법으로 증명해 봅시다.

$$ P(n):\quad 1 + 2 + 3 + \cdots + n = \frac{n(n+1)}{2} $$

🔍 단계별 증명 살펴보기

아래 각 단계를 클릭하면 자세한 설명을 볼 수 있습니다

  1. 📌 기초 단계 (Base Case) \(n = 1\): 좌변 = \(1\), 우변 = \(\frac{1(1+1)}{2} = 1\) \(1 = 1\) 이므로 \(P(1)\)은 참입니다. ✅
  2. 📌 귀납 가정 (Inductive Hypothesis) \(n = k\)일 때 성립한다고 가정: \(1 + 2 + \cdots + k = \frac{k(k+1)}{2}\) 이等式를 참이라고 가정합니다.
  3. 📌 귀납 단계 — \(P(k) \Rightarrow P(k+1)\) \(1 + 2 + \cdots + k + (k+1) = \frac{k(k+1)}{2} + (k+1)\) 양변에 \(k+1\)을 더합니다 (귀납 가정 활용).
  4. 📌 귀납 단계 — 정리 \(\frac{k(k+1)}{2} + (k+1) = \frac{k(k+1) + 2(k+1)}{2} = \frac{(k+1)(k+2)}{2}\) 우변을 정리하면 \(n=k+1\)일 때의 공식 \(\frac{(k+1)(k+2)}{2}\)과 일치! ✅
  5. 🎯 결론 (Conclusion) \(P(1)\) 참, \(P(k) \Rightarrow P(k+1)\) 참 → 모든 \(n \in \mathbb{N}\)에 대해 \(P(n)\) 참! 따라서 \(1 + 2 + \cdots + n = \frac{n(n+1)}{2}\)은 모든 자연수 \(n\)에 대해 성립합니다. 🎉
기초 단계: 가장 작은 자연수 \(n=1\)일 때 주어진 공식이 성립하는지 직접 확인합니다.
좌변: \(1\) / 우변: \(\frac{1 \cdot 2}{2} = 1\) → 일치하므로 \(P(1)\)은 참입니다.
